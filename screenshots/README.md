# App Screenshot Placeholder

## How to Add Your App Screenshot

1. **Take a screenshot of your app:**
   - Run the app: `npm run dev`
   - Open in browser (preferably mobile view or actual device)
   - Take a screenshot showing the main interface

2. **For Desktop Screenshot:**
   - Open browser DevTools (F12)
   - Toggle Device Toolbar (Ctrl+Shift+M / Cmd+Opt+M)
   - Select a mobile device (e.g., iPhone 12 Pro)
   - Capture the full app interface

3. **For Actual Mobile Screenshot:**
   - Deploy to HTTPS (Vercel, Netlify, etc.)
   - Open on your iPhone
   - Take screenshot (Power + Volume Up on Face ID iPhones)
   - Transfer to computer

4. **Save the image:**
   - Rename to `app-preview.png`
   - Place in this `screenshots/` directory
   - Recommended size: 1200x800 pixels or similar aspect ratio

5. **Alternative formats supported:**
   - PNG (recommended for quality)
   - JPG/JPEG (smaller file size)
   - WebP (modern format)

## Example Screenshot Composition

A good screenshot should show:
- The volume meter visualization
- The threshold slider
- The Start/Stop button
- The settings panel (message input)
- Clean, uncluttered browser window
